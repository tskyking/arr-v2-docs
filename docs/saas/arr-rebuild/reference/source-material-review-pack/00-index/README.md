# ARR Source Material Review Pack

This folder is a first-pass public/reference pack for ingestion-tool planning.

## Purpose
Reduce the burden on Brian by collecting public/sample/vendor references and organizing them into a reviewable structure.

## What this is
- a curated reference pack
- vendor/public examples and docs
- notes about what kinds of fields/patterns each source may help test

## What this is not
- a replacement for Brian's real-world examples
- a perfect source of anonymized enterprise contract/accounting edge cases
- a final training set

## Best use
Brian can review these examples and comment on:
- which ones resemble real workflows he has seen
- which fields/patterns matter most
- what is missing from the public/vendor examples
- where enterprise nuance is still not represented

## Folder guide
- `01-quickbooks/` — QuickBooks-oriented references
- `02-xero/` — Xero-oriented references
- `03-netsuite/` — NetSuite-oriented references
- `04-chargebee/` — subscription billing/export references
- `05-tabs-market-reference/` — Tabs product/market comparison reference
- `06-contract-examples/` — notes about what contract examples to look for
- `99-notes/` — summary notes and next-step guidance
