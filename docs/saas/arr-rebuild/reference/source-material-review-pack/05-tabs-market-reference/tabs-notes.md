# Tabs Market Reference

## Why included
Tabs is relevant as a market/product comparison reference because it appears to market an AI-assisted contract-to-cash workflow covering contract ingestion, billing, revenue recognition, and reporting.

## What to compare against Brian's tool
- AI-assisted ingestion vs rules-only ingestion
- contract-to-cash breadth vs focused ARR/revenue analytics
- invoice automation and AR workflow
- revenue recognition flexibility
- visibility into ARR / renewals / what was invoiced vs collected vs recognized

## Important note
This folder is for product comparison and feature framing, not for any claim of code overlap.
