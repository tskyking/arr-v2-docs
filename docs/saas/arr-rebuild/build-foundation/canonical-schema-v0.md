# Canonical Schema v0

## Core entities
- Logo
- Site
- Customer account identifiers
- Source import
- Contract
- Contract line / SKU
- Billing schedule / milestones
- Revenue recognition method
- Cancellation / early-out terms
- Manual overrides
- Audit log

## Important concepts
- logo/parent-customer vs site/local-entity hierarchy
- contract scope: site-specific, multi-site, enterprise-wide
- ARR policy is configurable
- monthly override capability
- source lineage preserved
- ambiguity/review workflow built in
