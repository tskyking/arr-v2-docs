# MVP Scope v0

## In scope
- support first 1-2 source input patterns
- normalize to canonical schema
- review unresolved mappings / ambiguities
- load normalized data into ARR app
- calculate core ARR outputs
- support basic overrides and audit trail
- provide summary and drill-down reporting

## Out of scope initially
- every ERP/accounting integration
- full contract-to-cash automation
- broad AR/collections scope
- polished production infrastructure from day one
- complete parity with every legacy feature
